# game

2D Unity 6 (URP) проект.

## Быстрый старт (графика)

1. Откройте проект в Unity.
2. **Game → Import → Import All Art Packs** — RPG_Hero, 32rogues, тайлсет.
3. **Game → Setup** — пункты 1–5 (шаблон уровня, префабы).
4. Tile Palette: папка `Assets/Graphics/Environment/Tiles/Dungeon`.
5. Подробности: [Assets/Docs/GRAPHICS_SETUP_RU.md](Assets/Docs/GRAPHICS_SETUP_RU.md)

## Сюжетные сцены — меню «Game → Story»

Все сцены истории («Путь через Лес Судеб») собираются и рисуются автоматически через меню
**Game → Story**. Запускать пункты нужно в режиме **редактирования** (НЕ во время Play).

1. **Game → Story → 1. Build Slice Scenes** — создаёт/пересобирает все сцены истории
   (Поляна → Распутье → 3A/3B/3C → Перекрёсток → 5A/5B/5C → Коридор → Сердце Леса → Концовка):
   игрок, камера, Файрен, выборы, переходы, враги, концовки + прописывает их в Build Settings.
2. **Game → Story → 2. Paint Forest Tiles** — рисует тайлами все сцены **так, как они сохранены**
   (из слепков в `Assets/Scenes/Story/Captures/`). Безопасно перезапускать.

**Чтобы всё собралось и нарисовалось:** нажми **1. Build**, затем **2. Paint**.

Важно:
- Сцены в репозитории **уже нарисованы** — можно просто открыть `Assets/Scenes/Story/` и играть.
  Build → Paint нужен только чтобы пересобрать всё заново (рисунок сцен восстановится из слепков).
- Меню `MainMenu` → кнопка «Играть» ведёт на первую сцену.
- Перед запуском пунктов меню убедись, что игра **не запущена** (кнопка ▶️ не активна).

## Структура

- `Assets/Graphics/` — спрайты, анимации, контроллеры
- `Assets/Data/` — ScriptableObject (CharacterVisualData и др.)
- `Assets/Prefabs/` — префабы персонажей, UI, окружения
- `Assets/Scripts/` — код (`Data`, `Visual`, `UI`, `Editor`)
- `Assets/Scenes/Story/` — сюжетные сцены (+ `Captures/` — слепки ручной отрисовки)
