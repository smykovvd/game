# Исходники арт-паков

Скопировано из корня репозитория:

- `32rogues/` — NPC, предметы, тайлы 32×32 (см. `*.txt` для координат спрайтов)
- `P_P_FREE_RPG_TILESET/` — подземелье/остров 24×24
- `RPG_Hero/` — анимации игрока 40×40 (лежит в `Characters/Player/Sprites/RPG_Hero`)

## Импорт в Unity

Меню **Game → Import → Import All Art Packs**.

Лицензия 32rogues: см. `32rogues/LICENSE.txt`.
